Shared honesty contract (everything obeys it)

  • Chat owns the screen; status collapses to reclaim space; preferences persist per project.
  • Proven ≠ inferred. Estimates live in a dashed “guesses, not facts” area and can never turn a tile green no ETA · no fake progress
  • Unknown = gray, never green. Every state carries an icon + word, not color alone.
  • Evidence is bound to the exact saved version + time; “Done” = proof-matched-stakes (+ your OK), not “worker stopped.”
  • The live feed is honestly labelled where it isn’t wired yet — no faked progress bar.
